# Agent Skills Repo

- **Read `WRITING-GUIDE.md` before authoring or editing a skill or agent.**
- **When updating a shared schema in `references/`, follow `WRITING-GUIDE.md` → Shared schema workflow.**
- **One decision at a time.** When walking design decisions, present one at a time: mechanism → problem → proposed fix → one question. Wait for the answer. Multi-select only for co-decided sub-options within a single fix.
- **Multi-reviewer findings: filter by ANY, not ALL.** Surface findings where at least one reviewer crossed the threshold. Show max confidence and which reviewers crossed (e.g., "0.85 / 0.30 — A1 only"). Walk sequentially.
